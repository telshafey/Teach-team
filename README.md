# Bokra Team

منصة متكاملة لإدارة فرق العمل، تتيح تخطيط المشاريع، تسجيل المهام وساعات العمل، إدارة الشؤون المالية، وعقد الاجتماعات الافتراضية، معززة بتقارير وملخصات ذكية.

## الميزات الرئيسية

- **إدارة المشاريع والمهام:** تخطيط المشاريع، إنشاء المهام، وتتبع التقدم.
- **تسجيل الوقت:** تسجيل ساعات العمل بدقة لكل مهمة ومشروع.
- **الإدارة المالية:** تتبع المصروفات، الرواتب، وتكاليف المشاريع.
- **اجتماعات افتراضية:** عقد اجتماعات فيديو مدمجة داخل المنصة.
- **تقارير وتحليلات:** الحصول على رؤى ذكية حول أداء الفريق والمشاريع.
